Vous êtes invités au tout premier Code Retreat de Montpellier !

Un Code Retreat une journée intense de code, mettant le focus sur les principes fondamentaux de développement logiciel. Les programmeurs de tous les langages sont invités - ça permet de brasser et challenger les idées. Six séances de 45 minutes permettent de vous exercer en paire dans un environnement sans contrainte. Après 45 minutes, le code est effacé avant de préparer la session suivante. On débriefe. On change de binôme (et si possible de langage de programmation). Et on recommence.

Chaque séance est un challenge différent. C'est l'occasion idéale pour tester de nouvelles pratiques, expérimenter de nouvelles technos. La force du groupe est sa diversité, c'est pourquoi nous avons besoin de vous.

Si vous êtes intéressé, vous n'avez besoin que de votre portable avec votre (ou vos nombreux) environnement de développement. La préférence est donnée au TDD, alors un environnement de test est indispensable.

L'évènement a lieu le 24 mars, et s'inscrit dans le mouvement du Software Craftsmanship sudiste en pleine croissance !
